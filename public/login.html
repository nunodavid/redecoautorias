The tool Rede de Coautorias is meant for the analysis of scientific collaborations using data from CiÃªncia-IUL. Two main options are available: Co-authorship network and Community detection.

The co-authorship network option is a tool for designing and exploring a network, whose nodes represent authors and edges represent scientific publications between authors.

The Community detection option is a tool for producing and exploring co-authorship communities, based on the co-authorship network. For community detection two algorithms are available: ABCD and MCL.
